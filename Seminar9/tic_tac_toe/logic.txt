Подготовка:
- заполняем массив ходов пробелами +
- создаем массив с выигрышными комбинациями +
- получаем имена игроков +
- определяем, кто ходит первым - это крестик +
Основной цикл:
    - Очищаем экран +
    - Рисуем поле +
    - Даем ход игроку +
    - Спрашиваем в какую позицию поставить крестик +
    - Если поле не занято, ставим крестик,
      иначе: просим сделать допустимый ход  +
    - Записываем крестик в массив ходов +
    - Передаем ход +
